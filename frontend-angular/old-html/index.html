Projeto + Sangue

Projeto realizado pela Microsoft em parceria com a PUCRS no Students to business 2018/2 trilha sistemas web com angular/node e javascript

Projeto que visa conectar aqueles que precisam de sangue em POA com aqueles que tem sangue e ideais altruístas de forma mais pessoal: o sistema desenvolvido em Javascript faz com que o conceito de doar sangue seja extendido de pessoa para pessoa e não de pessoa a bancos de sangue, demonstrando o perfil de quem vai receber o sangue e localização de acordo com a compatibilidade de sangues. Uma rede social com objetivos altruístas, afinal, 1% dos brasileiros doam sangue anualmente: lamentável.
